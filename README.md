# 🏠 House Price Prediction using Logistic Regression

## 📌 Overview

This project uses Logistic Regression to classify houses into price categories such as *affordable* or *expensive*. Instead of predicting exact prices, it predicts the probability of a house belonging to a specific price range based on its features.

---

## 🚀 Features

* Data preprocessing and cleaning
* Feature engineering
* Logistic Regression model implementation
* Model evaluation (Accuracy, Confusion Matrix, Precision, Recall)
* Simple and beginner-friendly code

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

---

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/house-price-prediction.git
cd house-price-prediction
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the main script or notebook to train and test the model:

```bash
python main.py
```

---

## 📊 Model Details

* Algorithm: Logistic Regression
* Input Features: Location, Size, Bedrooms, Bathrooms, Amenities
* Output: Price category (Affordable / Expensive)

---

## 📈 Evaluation Metrics

* Accuracy
* Confusion Matrix
* Precision
* Recall

---

## 📌 Use Case

* Classifying houses based on price range
* Real estate market analysis
* Assisting buyers and sellers in decision-making

---

## ⚠️ Note

Logistic Regression is used for classification in this project. For predicting exact house prices, regression models like Linear Regression or Random Forest are more suitable.
